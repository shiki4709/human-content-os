# Human — Content OS

AI-powered content OS. Turn any source into platform-native posts for LinkedIn, X, Substack, and Rednote 小红书.

## Stack
- Next.js 14 · Supabase · Tailwind · Anthropic Claude API

## Setup

```bash
npm install
cp .env.example .env.local
# Fill in your API keys
npm run dev
```

## For AI agents
Read `CLAUDE.md` first — it contains the full build spec, database schema, phase order, and all product rules.

The complete UI prototype is in `prototype.html` — open it in a browser to see exactly what to build.
